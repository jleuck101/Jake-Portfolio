Read Node — Repath Toolkit

Nuke Media Pipeline

Read-node hotbox utilities for fast repathing and sequence refresh during transfers and shot updates.

  • Replace All File Paths: selects all Reads/Writes and opens Nuke’s filename search/replace for global repaths
  • Frame Range to Files: rebuilds the Read path using the detected sequence + frame range from disk
  • Designed to reduce “missing Read” cleanup time when moving between machines/drives

Read Node — Timing / Alignment

Nuke Timing Workflow

Quick alignment helpers for setting Read start frames consistently across a comp.

  • At Comp Start: sets selected Reads to start at the script’s first frame
  • At Current Playhead: sets selected Reads to start at the current timeline frame
  • Supports batching across multiple selected Reads to keep timing consistent

Transform — Concatenate / Split Operations

Nuke Transforms Cleanup

Hotbox utilities for flattening multi-node Transform chains into one node, or breaking a single Transform into readable, operation-specific nodes.

  • Concatenate Transforms: selects a straight chain of Transform nodes and bakes the combined result into the last node (then rewires and deletes upstream nodes)
  • Split Transform Operations: splits translate/rotate/scale/skew (and optional center) into separate Transform nodes, preserving animation/expressions and skew order
  • Temporarily disables Auto-Key to avoid creating unwanted keys on the current frame
