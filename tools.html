Autocrop — Range + Padding

Nuke Crop Workflow

Autocrop over a custom frame range, then keep the animated crop curve while adding a padding slider for safe comps.

  • Custom panel: First / Last / Increment, Layer, and total Padding
  • Batch-friendly: runs per selected node and creates a Crop node each time
  • Adds a Padding (px) knob and expands the crop box via expressions so the original autocrop animation is preserved
Preview

Transform — Concatenate / Split Operations

Nuke Transforms Cleanup

Pair of hotbox utilities for breaking a Transform into readable per-operation nodes, then recombining them back into a single Transform.

  • Split Transform Operations: splits translate/rotate/scale/skew (and optional center) into separate Transform nodes while preserving animation/expressions and skew order
  • Concatenate Transforms: recombines a straight chain back into one Transform (non-destructive for split-generated chains)
  • Temporarily disables Auto-Key to avoid creating unwanted keys
Preview

Roto — Split Shapes into Nodes

Nuke Roto Organization

Converts a dense Roto/RotoPaint into multiple nodes so shapes can be isolated for handoff, reuse, troubleshooting, or performance-heavy comps. Inspired by a similar tool from Alex Villabon.

  • Creates one new node per shape (All), per selected shape, or per top-level folder (Layers mode)
  • Copies the original node’s settings and preserves animation/expressions where possible
  • Auto-labels new nodes using the shape or folder name (and keeps any root-level shapes grouped as RootShapes)

Backup Node — Safety Snapshot

Nuke Safety Workflow

One-click checkpoint before risky edits (tracking tweaks, keyframe cleanup, or heavy node refactors).

  • Duplicates the selected node(s) into a _Backup copy for quick rollback
  • Disconnects all inputs so the backup can’t affect the comp
  • Disables the backup when possible, offsets it in the DAG, and restores the Viewer input if it was temporarily hijacked
